# Changelog v0.5.0 — Engines Architecture

**Data:** 2026-06-19
**Ciclo:** 05

## Novas Features

### 🏗️ Engine Interfaces (shared/)
- **IGraphicsEngine** (#64): sprite/animation, camera, particles, layers, effects
- **IGameplayEngine** (#65): combat, skills, classes, buffs, loot, inventory, movement, collision
- **IStoryEngine** (#66): quest lifecycle, dialogue trees, quest log, NPC interactions
- **IMapEngine** (#67): map loading, collision grid, portals, spawn points, map editor

### 🎨 GraphicsEngine (client)
- Implementação concreta da IGraphicsEngine com PixiJS
- Camera com follow/snap/screenToWorld/worldToScreen
- Sprites com SimpleSprite wrapper
- Damage numbers com drift + fade (showDamageNumber)
- Suporte a layers (RenderLayer enum)

### ⚔️ GameplayEngine (server)
- Implementação concreta da IGameplayEngine com injeção de dependência
- Wraps CombatSystem, MovementSystem, CollisionSystem
- Stubs de skills, buffs, loot, inventory (contratos prontos)

## Bugs Corrigidos
- **#62** — ENTITY_UPDATE nunca enviado pelo servidor (agora broadcast a cada tick para entidades modificadas)
- **#63** — PLAYER_ATTACK não fazia broadcast (agora usa world.broadcastToMap + ENTITY_UPDATE do alvo)

## Arquitetura
- Interfaces PURAS em shared/ (sem runtime dependencies, `import type`)
- Strangler Fig pattern — código legado intacto, engines como camada superior
- Injeção de dependência nas engines concretas
